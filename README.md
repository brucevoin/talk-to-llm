# talk-to-llm

This project aims to build a tool to talk with your local LLM(Ollama) smoothly.

## involled technicals

### ASR

- portaudio
- pyaudio :for recording audio
- whisper : for automatic speech recognition

### Local LLMs

- Ollama

### TTS

- xtts_v2(include voice cloning)
- Or EmotiVoice from netease

## Develop

- python 3.10.13
